# Framework Playground

Interactive code playground for testing and comparing JavaScript frameworks side-by-side.

## What it does

Write code in Vanilla JS, React, Vue, Svelte, or Angular and see the output live. Auto-run updates the preview as you type, or turn it off and run manually.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Frameworks

- Vanilla JavaScript
- React 18 (with Babel)
- Vue 3
- Svelte 5 (with compiler)
- Angular (TypeScript transpilation)

All frameworks run in an isolated iframe using CDN resources.
